# 🦜 OpenMacaw

**The AI agent that natively parrots your workflows on macOS.**

OpenMacaw is an upcoming macOS-native fork of the popular OpenClaw agent. 

### Why OpenMacaw?
While OpenClaw is a fantastic cross-platform general computing agent, macOS users deserve a first-class, native experience. 

**Planned Features:**
* 🍎 **Apple Silicon Native:** Optimized to run local LLMs via MLX.
* 🖥️ **SwiftUI Frontend:** A beautiful, lightweight Mac-native interface instead of heavy electron/web wrappers.
* 🤖 **The "Parrot" Engine:** Demonstrate a workflow using macOS Accessibility APIs, and OpenMacaw will literally *parrot* it back to automate your repetitive tasks.
* ⚡ **Deep macOS Integration:** Hooks directly into Apple Shortcuts and Automator.
